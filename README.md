# SkillCensus — India's Open Human Talent Network

> Every skill. Every district. Findable.

**SkillCensus** makes India's 450 million skilled workers discoverable — welders, coders, nurses, artisans — no middlemen, no fees, ever.

## Features

- 🗺️ **Real-Time Talent Map** — Live Leaflet map showing skill clusters across every district
- 💬 **Built-in Secure Chat** — Connect and communicate directly on-platform
- ✅ **Skill Verification** — Government certificates, peer endorsements, employer references
- 📊 **Personal Dashboard** — Track profile views, requests, and skill demand trends
- 🌐 **Works for Everyone** — Optimized for 2G, designed for the 95% LinkedIn never reached
- 🔒 **Privacy First** — Phone number stays hidden until YOU choose to share

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Maps**: Leaflet.js + OpenStreetMap via CARTO dark tiles
- **Fonts**: Cabinet Grotesk, JetBrains Mono, Instrument Serif

## Quick Start

```bash
# Just open the file
open index.html

# Or serve locally
python -m http.server 8080
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import project
3. Select this repo → Deploy
4. Your site is live at `your-project.vercel.app` 🎉
